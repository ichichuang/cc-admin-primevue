vue-echarts 插件的 完整配置项清单，以便进行二次封装。以下是该插件支持的所有配置项、功能和事件的清单：

1. 基础配置
1.1 options

完整的 ECharts 配置对象，支持所有 ECharts 配置项。

1.2 theme

theme: 设置主题，可以选择内置的主题（如 light、dark），也可以自定义主题。

1.3 renderer

renderer: 设置渲染方式，支持 canvas 或 svg。

1.4 autoResize

autoResize: 是否开启自动调整图表大小（默认开启）。支持 true | false。

1.5 style

style: 自定义图表样式配置，例如宽高、背景颜色等。

示例: { width: '100%', height: '400px' }

2. 图表配置项
2.1 title

title: 图表标题的配置。

text: 标题文字。

subtext: 子标题文字。

left、right、top、bottom: 控制标题位置。

2.2 tooltip

tooltip: 提示框配置，展示数据时的提示。

trigger: 触发类型，支持 item、axis、none。

formatter: 自定义格式化内容。

2.3 legend

legend: 图例配置，展示系列的名称。

data: 图例项的名称。

orient: 图例布局方式，支持 horizontal | vertical。

top、left、right、bottom: 控制图例位置。

2.4 xAxis & yAxis

xAxis、yAxis: 坐标轴配置项。

type: 坐标轴类型，category、value、log 等。

data: 类目轴的类目数据。

min、max: 设置坐标轴最小/最大值。

axisLabel: 坐标轴标签的格式化设置。

2.5 series

series: 数据系列配置，包含多个数据项，可以是柱状图、折线图等。

type: 系列类型，如 line、bar、pie、scatter、radar 等。

name: 系列名称。

data: 数据数组。

color: 数据系列的颜色。

itemStyle: 自定义图形样式。

2.6 grid

grid: 图表网格配置，控制图表的布局与间距。

top、left、right、bottom: 控制网格的边距。

containLabel: 是否包含标签。

2.7 animation

animation: 开启/关闭动画。

duration: 动画时长。

easing: 动画缓动效果，如 linear、easeIn、easeOut。

3. 事件配置
3.1 @ 绑定的事件

@click: 图表点击事件。

@dblclick: 图表双击事件。

@mouseover: 鼠标悬停事件。

@mouseout: 鼠标移出事件。

@legendselectchanged: 图例选择变化事件。

@datazoom: 数据缩放事件。

3.2 onEvents

onEvents: 将事件映射为 Vue 事件监听器。

格式：{ event: 'click', handler: this.handleClick }

4. 数据与交互
4.1 dataZoom

dataZoom: 数据缩放组件，用于图表数据范围选择。

type: 缩放类型，支持 inside 或 slider。

start、end: 缩放的起始和结束位置（百分比）。

4.2 brush

brush: 选择区域的配置，支持选择图表区域进行放大、选中等操作。

4.3 visualMap

visualMap: 可视化映射组件，用于将数据映射到不同的视觉元素（如颜色、大小）。

min、max: 数据值的最小值和最大值。

dimension: 映射的维度。

5. 其它配置项
5.1 backgroundColor

backgroundColor: 设置图表的背景颜色。

5.2 markPoint

markPoint: 图表的标记点，通常用于标记特殊数据点。

symbol: 标记点的图形类型。

data: 标记点的数据。

5.3 markLine

markLine: 图表的标记线，通常用于参考线。

data: 标记线的数据，支持设置起点和终点。

5.4 legendHoverLink

legendHoverLink: 是否开启图例的悬停联动效果。

true | false

5.5 toolbox

toolbox: 工具箱组件，通常用于启用图表的一些交互功能，如数据缩放、保存图片等。

show: 是否显示工具箱。

feature: 工具项配置，如 saveAsImage、restore、dataView 等。

5.6 titleStyle

titleStyle: 配置标题的字体样式。

5.7 axisPointer

axisPointer: 坐标轴指示器，用于展示当前坐标系的指示器。

show: 是否显示指示器。

type: 指示器类型，line、shadow 等。

6. 支持的图表类型

line: 折线图。

bar: 柱状图。

pie: 饼图。

scatter: 散点图。

radar: 雷达图。

funnel: 漏斗图。

gauge: 仪表盘。

heatmap: 热力图。

treemap: 树图。

sankey: 桑基图。

map: 地图。

7. 高级功能
7.1 rich

rich: 富文本样式，可以为文本标签、轴标签等提供更多的样式设置。

7.2 series.tooltip

tooltip: 每个系列的提示框配置。

trigger: 触发类型，支持 item、axis、none。

formatter: 自定义格式化提示内容。

7.3 textStyle

textStyle: 设置文本的字体、颜色、大小等。

7.4 lineStyle

lineStyle: 设置线条的颜色、宽度、类型（如 solid、dashed）等。

7.5 itemStyle

itemStyle: 设置图形元素的样式，如颜色、边框等。

8. 版本管理与更新
8.1 echartsInstance

echartsInstance: 获取 ECharts 实例，用于调用 ECharts 的 API，例如 echartsInstance.setOption() 等。

8.2 @load

@load: 图表加载完成后的事件。

9. 性能优化配置

lazyLoad: 图表是否支持懒加载，减少初始渲染的性能消耗。

这份清单包含了 vue-echarts 插件的几乎所有功能配置项和事件
